# ドット魂 〜昭和激闘伝〜 (DOT DAMASHII)

昭和レトロ風・ドット絵2D対戦格闘ゲーム。ブラウザだけで動きます（ビルド不要・依存ゼロ）。

![genre](https://img.shields.io/badge/genre-%E5%AF%BE%E6%88%A6%E6%A0%BC%E9%97%98-red)

## 遊び方

`index.html` をブラウザで開くだけ。

```
start index.html        # Windows
```

### モード

- **ひとりであそぶ** — VS COM戦
- **ふたりであそぶ** — ローカル2P対戦（1枚のキーボードを共有）
- **ネットたいせん** — インターネット越しのオンライン1対1対戦

### オンライン対戦のやり方

1. ホスト側: タイトルで「ネットたいせん」→「へやを つくる」→ 表示された**4桁のあいことば**を相手に伝える（LINEでも電話でもOK）
2. ゲスト側: 「ネットたいせん」→「あいことばで さんか」→ 4桁を入力してENTER
3. 接続したら各自キャラを選んで対戦開始！**オンライン中は両者とも1P操作（A/D/W/S/J/K/L）**

技術メモ: WebRTC（PeerJS）によるP2P直接通信。入力ビットマスクを毎フレーム交換する遅延4フレームの決定論ロックステップ方式で、60フレームごとに状態ハッシュを照合して同期ずれを監視します。対戦データはサーバーを経由しません（接続仲介のみPeerJSクラウドを利用）。

### 操作

| 操作 | 1P | 2P |
|------|----|----|
| 移動 | A / D | ← / → |
| ジャンプ | W | ↑ |
| ガード | S | ↓ |
| 弱攻撃 | J | , |
| 強攻撃 | K | . |
| 必殺技 | L | / |

- 必殺技は画面下の**ゲージが満タン（必殺OK!）**のときだけ発動可能
- ゲージは攻撃を当てる/食らう/ガードすると溜まる
- 2ラウンド先取で勝利。60秒のタイムアップ時は残り体力勝負

## 技術構成

- 素のHTML/CSS/JavaScript（フレームワーク・画像ファイル不使用）
- ドット絵は**テキストグリッド＋パレット**でデータ化し、Canvasに展開（`js/sprites.js`）
- BGM/SFXはWebAudioでリアルタイム生成するチップチューン（`js/audio.js`）
- ステージ背景もすべてプログラム生成（`js/stages.js`）
- CRT風スキャンライン＋ビネット演出（`style.css`）

## 開発者向け

### キャラクターの追加

1. `SPEC.md` の仕様に従って `js/characters/<id>.js` を作成
2. 検証: `node tools/validate.js js/characters/<id>.js`（PASSするまで修正）
3. プレビュー: `node tools/preview.js js/characters/<id>.js idle 0`
4. `index.html` に `<script>` タグを追加

### ファイル構成

```
index.html        エントリポイント
style.css         CRT演出・レイアウト
SPEC.md           キャラクターデータ仕様書
js/
  main.js         ゲーム状態管理・60fpsループ・ネット対戦ロックステップ
  net.js          WebRTC通信モジュール（PeerJS・部屋コード・入力同期）
  lib/peerjs.min.js  PeerJS本体（同梱）
  engine.js       Fighter/Projectile/Battle（戦闘コア）
  ai.js           CPU対戦AI
  ui.js           HUD・タイトル・選択・リザルト画面
  sprites.js      テキストグリッド→Canvasスプライト変換
  stages.js       ステージ背景のプログラム生成
  audio.js        WebAudioチップチューン（BGM/SFX）
  fx.js           パーティクル・画面揺れ
  input.js        キーボード入力
  characters/     キャラクターデータ（4体）
tools/
  validate.js     キャラデータ検証
  preview.js      ドット絵テキストプレビュー
```
